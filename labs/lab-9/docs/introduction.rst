Introduction
============
This project is a web application that allows users to sign up, log in, and view user information. It is built using Flask for the backend, SQLAlchemy as the ORM for database interaction, and PostgreSQL as the database. The application includes form validation, password hashing with bcrypt, and basic logging functionality to track user activity and errors.
