1. Introduction¶

This documentation covers the CMPT 221 lab assignments. The software development course teaches concepts like creating pull requests, branching, and merging using Git and GitHub. Each lab assignment focuses on different aspects of software development, including databases, python, pytest, html, css, testing, and many more.
